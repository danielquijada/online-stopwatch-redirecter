Online-Stopwatch Redirecter

Simple tool to redirect to Online-Stopwatch

You can either input the time tag in the input field below, or add it as a URL parameter

Example: danielquijada.github.io/online-stopwatch-redirecter/?timer=10m15s

Supported times (accepted keywords):

  • Seconds (numbers with any non-accepted keywords or without keywords will default to seconds)
  • Minutes (minutes/minute/mins/min/m)
  • Hours (hours/hour/hrs/hr/h)
  • Days (days/day/d)
  • Weeks (weeks/week/w)
  • Months (31 days) (months/month/mon)
  • Years (years/year/yrs/yr/y)
